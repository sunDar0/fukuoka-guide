# 후쿠오카 여행 가이드 🇯🇵

후쿠오카 여행을 위한 인터랙티브 웹 가이드입니다. 여행 일정, 예산, 관광지 정보를 한눈에 확인하고 맞춤형 여행 계획을 생성할 수 있습니다.

## 주요 기능

### 맞춤형 여행 플래너
- 여행 날짜 및 인원(성인/아동/유아) 입력으로 자동 일정 생성
- 2박~7박 일정별 시간대별 상세 스케줄
- 예산 계산기 (숙박/교통/식비/관광 항목별 산출)
- 날씨 정보 및 짐 싸기 팁
- 공유 링크 생성

### 인터랙티브 지도
- Leaflet 기반 58개 관광 명소 + 25개 맛집 + 25개 숙소 지도
- 8개 지역별 필터링 (하카타, 텐진/나카스, 다자이후, 후쿠오카 시내, 이토시마, 키타큐슈, 유후인, 벳푸)
- 레이어 토글 (관광지/맛집/숙소 각각 표시/숨기기)
- 추천 경로 보기 (Nearest Neighbor 알고리즘 기반 동선 최적화)
- 명소 검색 및 Unsplash 이미지 갤러리

### 맛집 가이드
- 25개 맛집 데이터 (라멘, 모츠나베, 교자, 야키토리, 스시, 카페 등)
- 카테고리별 필터링
- 대표 메뉴, 가격대, 영업시간 정보

### 교통 가이드
- 공항→시내 이동 수단 비교 (지하철/버스/택시)
- 5개 교통 패스 상세 비교 (시티패스, SUNQ, JR패스 등)
- 여행 일수 기반 패스 추천
- 6개 주요 지역 간 이동 방법 (시간/비용/설명)

### 숙소 가이드
- 25개 숙소 데이터 (호텔/료칸/게스트하우스)
- 유형별/예산별 필터링 (절약형/중급/프리미엄)
- 시설 특징, 근처 관광지 연결 정보

### 관광지 사진 갤러리
- Unsplash API 연동 관광지/맛집/숙소 이미지
- Lazy loading 및 폴백 처리

### 오프라인 지원 (PWA)
- Service Worker 기반 오프라인 캐싱
- 모바일 홈 화면에 앱으로 설치 가능

### 모바일 최적화
- 지도 페이지 바텀시트 UI (터치 드래그)
- 하단 고정 네비게이션 바
- 터치 타겟 44px 보장

## 기술 스택

| 구분 | 기술 |
|------|------|
| 마크업 | HTML5 |
| 스타일 | CSS3 (다크 테마, 글래스모피즘) |
| 스크립트 | Vanilla JavaScript (ES6+) |
| 지도 | Leaflet.js v1.9.4 + MarkerCluster |
| 차트 | Chart.js |
| 이미지 | Unsplash Source API |
| 폰트 | Noto Sans KR (Google Fonts) |
| 타일 | CartoDB |
| PWA | Service Worker + Web App Manifest |

빌드 도구나 npm 패키지 없이 순수 정적 파일로 구성되어 있습니다.

## 프로젝트 구조

```
fukuoka-guide/
├── index.html      # 메인 여행 플래너 페이지
├── map.html        # 인터랙티브 지도 페이지
├── style.css       # 공통 스타일시트 (2,492줄)
├── data.js         # 전체 데이터 (58 명소 + 25 맛집 + 25 숙소 + 교통)
├── planner.js      # 여행 플래너 로직
├── map.js          # 지도 + 경로 최적화 + 레이어 관리
├── sw.js           # Service Worker (오프라인 캐싱)
├── manifest.json   # PWA 매니페스트
├── icon-192.svg    # 앱 아이콘 (192x192)
└── icon-512.svg    # 앱 아이콘 (512x512)
```

## 실행 방법

별도의 설치 과정 없이 정적 파일 서버로 실행할 수 있습니다.

```bash
# Python
python3 -m http.server 8000

# Node.js
npx http-server

# 브라우저에서 http://localhost:8000 접속
```
