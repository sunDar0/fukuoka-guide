# 후쿠오카 여행 가이드 🇯🇵

후쿠오카 여행을 위한 인터랙티브 웹 가이드입니다. 여행 일정, 예산, 관광지 정보를 한눈에 확인하고 맞춤형 여행 계획을 생성할 수 있습니다.

## 주요 기능

### 맞춤형 여행 플래너
- 여행 날짜 및 인원(성인/아동/유아) 입력으로 자동 일정 생성
- 2박~7박 일정별 시간대별 상세 스케줄
- 예산 계산기 (숙박/교통/식비/관광 항목별 산출)
- 날씨 정보 및 짐 싸기 팁
- 공유 링크 생성

### 인터랙티브 지도
- Leaflet 기반 58개 관광 명소 지도
- 8개 지역별 필터링 (하카타, 텐진/나카스, 다자이후, 후쿠오카 시내, 이토시마, 키타큐슈, 유후인, 벳푸)
- 명소 검색 기능
- 클릭 시 운영시간, 입장료, 근처 숙소 등 상세 정보 확인

### 관광지 데이터
- 58개 명소의 상세 정보 (운영시간, 입장료, 소요시간, 웹사이트)
- 아이 동반 여행 적합 여부 표시
- 근처 추천 숙소 및 가격 정보
- 후쿠오카 대표 음식 6선 소개

## 기술 스택

| 구분 | 기술 |
|------|------|
| 마크업 | HTML5 |
| 스타일 | CSS3 (다크 테마, 글래스모피즘) |
| 스크립트 | Vanilla JavaScript (ES6+) |
| 지도 | Leaflet.js v1.9.4 + MarkerCluster |
| 차트 | Chart.js |
| 폰트 | Noto Sans KR (Google Fonts) |
| 타일 | CartoDB Dark Matter |

빌드 도구나 npm 패키지 없이 순수 정적 파일로 구성되어 있습니다.

## 프로젝트 구조

```
fukuoka-guide/
├── index.html      # 메인 여행 플래너 페이지
├── map.html        # 인터랙티브 지도 페이지
├── style.css       # 공통 스타일시트
├── data.js         # 관광지 데이터 (58개 명소, 8개 지역)
├── planner.js      # 여행 플래너 로직
└── map.js          # 지도 기능
```

## 실행 방법

별도의 설치 과정 없이 정적 파일 서버로 실행할 수 있습니다.

```bash
# Python
python3 -m http.server 8000

# Node.js
npx http-server

# 브라우저에서 http://localhost:8000 접속
```
